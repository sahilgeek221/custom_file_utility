# 📂 TidyDesk File Utility

A sleek, interactive desktop application built with Python and CustomTkinter. TidyDesk helps you instantly organize cluttered directories, safely merge files, and manage your file system with confidence using a built-in dry-run and undo engine.

## ✨ Features

* **Smart Organizer:** Automatically sorts files into `Images`, `Documents`, `Code`, `Archives`, and `Audio` folders based on their extensions.
* **File Merger:** Easily select multiple text, code, or CSV files and merge them into a single, clearly formatted output document.
* **Dry-Run Engine:** Preview exactly which files will be moved before any changes are made to your hard drive.
* **History & Undo:** Made a mistake? The app logs your actions and features a one-click "Undo" button to restore files to their original locations.
* **Modern Custom UI:** Fully translucent dark/light mode interface with a built-in color wheel to customize the app's theme on the fly.
